# 광명빌 소개 사이트

## 주택 정보

- 주소 : 전북 전주시 덕진구 건지2길 9
- 네이버 지도 링크 : https://naver.me/xUZDVqR2
- 층 구성 : 총 4개층. 1층은 의료기 상가. 2~4층은 원룸/투룸
- 세대 구성 : 총 8세대로 원룸 5개, 투룸 3개
- 특징
  - 전북대병원 바로 앞. 정문에서 도보 2분 거리
  - 2026년 3월에 리모델링 공사가 끝난 초 신축
  - 화이트 톤 위주로 인테리어 진행
  - 에어컨, 냉장고, 세탁기, 인덕션, 옷장 등 모두 새거
  - 건물 입구부터 각 층별로 CCTV 설치가 되어 있어 안전 보장
  - 전문 관리인 체계로 빠르게 민원 및 불편 사항 대응 가능

## 소개 사이트 구성

- 세대별 소개는 이미지 위주로 설명
- 각 세대별로 지원되는 가전, 가구는 깔끔하게 아이콘 형태로 표시
- 네이버 지도 SDK를 활용해서 약도 표시
- 사이트 전반적으로 신축이라는 느낌이 나게끔 밝고 모던한 디자인 톤앤매너 적용

## 구현시 참고 사항

- 세대별 사진은 4월초에 촬영 후 올릴 예정
